🧮 Calculator (Python)

📌 Overview
This is a simple Calculator application built using Python. The program performs basic arithmetic operations such as addition, subtraction, multiplication, and division based on user input.

This project helped me strengthen my understanding of Python fundamentals, user input handling, conditional statements, and functions.

---

⚙️ Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly interface
- Handles multiple calculations

---

🛠️ Tech Stack
- Python 3

---

🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/your-username/Calculator.git
```

2. Navigate to the project folder
```bash
cd Calculator
```

3. Run the program
```bash
python main.py
```
