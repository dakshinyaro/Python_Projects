📝 Quiz Application (Python Tkinter)

📌 Overview

This is a GUI-based Quiz Application developed using Python and Tkinter. The application presents multiple-choice questions to users, evaluates their answers, and displays the final score at the end of the quiz.

This project helped improve my understanding of Python GUI development, event handling, conditional logic, and user interaction.

---

⚙️ Features

* Interactive graphical user interface (GUI)
* Multiple-choice questions
* Score tracking system
* Instant navigation between questions
* Final score display
* Beginner-friendly design

---

🛠️ Tech Stack

* Python 3
* Tkinter

---

🚀 How to Run

1. Navigate to the project folder

```bash
cd QuizApplication
```

2. Run the application

```bash
python main.py
```
